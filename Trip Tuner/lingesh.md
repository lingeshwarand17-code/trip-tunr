# 🌍 Trip Tuner – Travel Booking Website

Trip Tuner is a full-stack travel booking web application that allows users to search, explore, and book travel packages online.  
This project includes both frontend and backend features with user authentication, booking system, and admin management.

---

## 🚀 Features

### 👨‍💻 Frontend Features

- User Login & Signup
- Home page with travel offers
- Search trips by location, price, and date
- Trip details page
- Booking form
- Wishlist / Favorite trips
- Payment page UI
- My bookings page
- User profile page
- Reviews & ratings
- Contact page
- FAQ page
- Responsive design (Mobile + Desktop)
- Dark / Light mode (optional)

---

### ⚙️ Backend Features

- User authentication (Login / Signup / Logout)
- Secure password encryption
- Database storage for users
- Store travel packages
- Booking management
- Payment processing (dummy / API)
- Review & rating system
- Wishlist API
- Search filter API
- Booking history
